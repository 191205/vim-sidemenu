# vim-sidemenu

> A small side-menu useful for Vim terminal users

## Install

Use your favorite package-manager.

## Setup

```vim
nmap <Leader>l <Plug>(sidemenu)
xmap <Leader>l <Plug>(sidemenu-visual)
```
